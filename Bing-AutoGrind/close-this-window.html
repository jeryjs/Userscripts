To enable closing windows not opened by script in Firefox, follow these steps:

  1. Open Firefox and type about:config in the address bar.
  2. Press Enter and accept the risk warning.
  3. Search for the preference named dom.allow_scripts_to_close_windows.
  4. Change its value to true.
